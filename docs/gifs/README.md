# Demo GIFs

Drop short demo GIFs here, then uncomment the matching rows in the main [README](../../README.md#demos).

## Suggested clips
| File | What to show | Keep it to |
|------|--------------|-----------|
| `fleet.gif` | Live fleet view — hierarchy, role tally, an alert firing | ~8–12s |
| `intel.gif` | Intel Tools — paste a d-scan, then the constellation map + a right-click → Dotlan/zKill | ~10s |
| `ping.gif` | Ping / MOTD — pick a doctrine + channels, MOTD updates, "Set on fleet" | ~10s |
| `overlay.gif` | The alert overlay floating over EVE during a fight | ~6–8s |

## How to record (Windows)
- **[ScreenToGif](https://www.screentogif.com/)** (free, open-source) is the easiest — record a region, trim, export as GIF.
- Record at the window's native size; crop to just the FCAT window (or the overlay over EVE).
- Keep each clip short and loopable. Aim for **< ~6 MB** per GIF so the README stays light — reduce by lowering FPS (10–15 is fine), trimming length, or limiting colors on export.

## Tips
- Use a throwaway/test fleet so no sensitive intel is shown.
- For the overlay clip, run EVE in **borderless / windowed fullscreen** so the overlay draws over it.
